Manual de GitHub

¿Qué es GitHub?

GitHub es una plataforma para alojar y gestionar código fuente utilizando el sistema de control de versiones Git. Permite a los desarrolladores colaborar, compartir proyectos y contribuir a código abierto.

Primeros Pasos

  1. Crear una cuenta en GitHub.
  2. Instalar Git en tu sistema.
  3. Configurar tu nombre de usuario y correo electrónico.
  4. Crear un nuevo repositorio.

Comandos Básicos de Git

Comando Descripción
git init Inicializa un nuevo repositorio de Git.
git clone Clona un repositorio existente.
git add . Añade todos los cambios al área de preparación.
git commit -m "mensaje" Crea un nuevo commit con los cambios agregados.
git push Envía los commits al repositorio remoto.
